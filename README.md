App-cloc-pm
===========

> Count Lines of Code

CPAN package for [CLOC](http://cloc.sourceforge.net/) utility.

To install, pray the mantra

    perl Makefile.PL
    make
    make test
    make install

For more information point your browser to [online docs](https://metacpan.org/pod/App::cloc).
